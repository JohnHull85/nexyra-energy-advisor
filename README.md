
# NEXYRA Energy Advisor (Streamlit Prototype)

## Quick Start
1) Optional: create a virtual environment.
2) Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3) Run the app:
   ```bash
   streamlit run app.py
   ```
